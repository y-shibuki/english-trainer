# ドキュメントインデックス

各ドキュメントに関連するキーワードをタグとして付与している。

| ドキュメント | 概要 | タグ |
|-------------|------|------|
| `基本設計書.md` | 本アプリケーションの基本設計書 | アーキテクチャ, ドメインモデル, 非同期処理, 問題形式 |
| `技術スタック.md` | 本アプリケーションの技術スタック | React, FastAPI, Python, PostgreSQL, Docker |
| `API仕様書.md` | REST APIのエンドポイント仕様（リクエスト/レスポンス例含む） | API, エンドポイント, Article, Question, 解答判定 |
| `開発フェーズ計画.md` | Phase 0〜4の開発フェーズ整理（FEモック優先） | フェーズ計画, MSW, モック, FE開発 |
