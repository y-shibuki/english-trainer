# 技術スタック

## 1. デプロイ基盤

### アプリケーション実行環境
- Render
  - Web Service：Backend API
  - Background Service：Generator（バッチ）
  - Static Site：Frontend

### データベース
- Neon（Serverless PostgreSQL）

---

## 2. フロントエンド

- React
- TypeScript
- Fetch API（または軽量HTTPクライアント）
- shadcn/ui + Tailwind

※ SPA構成（SSRは採用しない）
※ Render Static Siteとしてデプロイ

---

## 3. バックエンド

- Python 3.11+
- FastAPI
- Uvicorn（ASGIサーバ）
- Pydantic（バリデーション）
- SQLAlchemy（ORM）

---

## 4. Generator（AIバッチ）

- Python（Backendと同一リポジトリ）
- FastAPIとは別エントリーポイント
- cronまたは手動実行
- OpenAI APIクライアント

---

## 5. Crawler

- Python
- requests
- BeautifulSoup

---

## 6. データベース

- PostgreSQL（Neon）
- マイグレーション管理：Alembic

---

## 7. AI連携

- OpenAI API（軽量モデル想定）
- HTTPベース呼び出し

## 8. 開発基盤

- Docker（各プロセス個別Dockerfile）

- バッチ型非同期モデル
